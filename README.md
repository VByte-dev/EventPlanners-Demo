# Event Planners Demo Website

A simple, reusable demo website template designed for event planners to showcase their work online. This template provides a clean and professional layout to display services, portfolio, and contact information.

## Features

- Fully responsive and works on all devices  
- Clean, modern design that can be adapted for different needs  
- Easy to customize with your own content, images, and branding  
- Built to showcase your services and professional work effectively  
- Lightweight and fast-loading for a smooth user experience
